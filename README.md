## 《黑神话悟空》Steam评测数据分析项目

该项目聚焦《黑神话悟空》的Steam评测数据，运用Python数据科学工具栈（Pandas、Numpy、Matplotlib、Seaborn、jieba等），从**情感倾向、游戏时长、内容关键词、评测时间**等维度开展深度分析，全面洞察玩家对游戏的评价与行为特征。


## 核心分析模块

### 1. 评测情感分析
- 数据显示游戏正面评测占比高达83.75%，远高于Steam平台平均好评率（约70%），说明游戏在玩家群体中获得了广泛认可。
- 通过饼图、柱状图直观呈现正负情感评测的数量与比例差异，为游戏口碑定位提供量化依据。


### 2. 游戏时长分析
- 玩家平均游戏时长为9.76小时，中位数9.30小时，分布较为集中；最长游戏时长达127.2小时，体现部分玩家对游戏的深度探索与高回放价值。
- 结合箱线图、小提琴图分析不同游戏时长分组的评测情感分布，挖掘游戏时长与玩家满意度的关联。


### 3. 评测内容关键词挖掘
- **正面关键词**：“游戏”“好玩”“神话”“3A”“国产”等高频词，凸显玩家对游戏品质、题材、国产3A定位的认可。
- **负面关键词**：“boss”“闪退”“优化”“差评”等词汇，反映游戏在BOSS设计、性能优化等方面存在待改进点。
- 运用jieba分词与关键词权重分析，清晰定位游戏的核心亮点与争议焦点。


### 4. 评测时间分布分析
- 评测高度集中于2024年8月，结合游戏发布时间，可把握玩家评价的时间趋势，为游戏运营节奏提供参考。


## 技术栈
- 数据处理：Pandas（数据读取、清洗、分组聚合）、Numpy（数值计算）；
- 可视化：Matplotlib、Seaborn（饼图、柱状图、箱线图、小提琴图等多维度可视化）；
- 文本挖掘：jieba（中文分词、关键词提取）；
- 其他：正则表达式（文本清洗）、自定义颜色方案（提升可视化可读性）。


## 应用价值
- 为游戏开发者提供**玩家反馈的量化洞察**，明确优化方向（如性能优化、BOSS设计）；
- 为游戏运营团队提供**口碑传播与玩家行为的分析依据**，辅助制定运营策略；
- 作为游戏数据分析的实战案例，为数据科学爱好者提供方法论参考。
